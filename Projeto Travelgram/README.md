🌍 Travelgram

Projeto desenvolvido durante meus estudos na Rocketseat, com foco na construção de interfaces modernas, responsivas e bem estruturadas utilizando HTML e CSS.

📸 Sobre o projeto

O Travelgram é uma página de perfil de viagens que simula uma rede social voltada para compartilhamento de experiências ao redor do mundo.

A interface apresenta:

Perfil do usuário com foto e descrição
Informações como localização, países visitados e número de fotos
Grid de imagens com destinos turísticos
Layout limpo, moderno e responsivo
🚀 Tecnologias utilizadas
HTML5
CSS3
Flexbox
Grid Layout
🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de:

Praticar estruturação semântica com HTML
Trabalhar posicionamento com Flexbox e Grid
Melhorar a organização de CSS
Criar interfaces com padrão mais profissional
Evoluir na construção de layouts responsivos
