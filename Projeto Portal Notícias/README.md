# 📰 Portal de Notícias

Projeto desenvolvido durante as aulas da Rocketseat com foco em **HTML** e **CSS**, utilizando principalmente **CSS Grid** para estruturação de layout.

## 🚀 Objetivo

O objetivo deste projeto foi praticar a construção de interfaces modernas e responsivas, aplicando conceitos fundamentais de desenvolvimento front-end, como:

* Estruturação semântica com HTML
* Estilização com CSS
* Criação de layouts com **CSS Grid**
* Organização visual de conteúdo (notícias)

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* CSS Grid

## 🎯 Aprendizados

Durante o desenvolvimento deste projeto, foi possível:

* Entender melhor o funcionamento do **CSS Grid**
* Trabalhar com divisão de layout em áreas
* Melhorar a organização do código CSS
* Criar uma interface semelhante a portais de notícias reais

## 💡 Melhorias futuras

* Tornar o layout totalmente responsivo
* Adicionar interatividade com JavaScript
* Integrar com APIs de notícias
* Melhorar acessibilidade
