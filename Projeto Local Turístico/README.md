# 🌍 Projeto Local Turístico

Projeto desenvolvido com o objetivo de apresentar informações sobre um destino turístico, incluindo descrição, imagens e pontos de interesse.

Este projeto foi criado como parte dos estudos em **HTML e CSS**, com foco em estruturação de conteúdo e estilização de páginas web.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Git e GitHub

---

## 📚 Objetivo do projeto

Este projeto teve como objetivo:

- Praticar a estruturação semântica com HTML
- Trabalhar estilização com CSS
- Organizar layout visual
- Desenvolver boas práticas de organização de arquivos
- Criar um projeto para portfólio
