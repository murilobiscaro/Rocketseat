# 🍽️ Projeto Receitas

Este é um projeto simples desenvolvido durante meus estudos de **HTML e CSS**, com o objetivo de praticar a criação de páginas web e organização de conteúdo.

## 📚 Sobre o projeto

A página apresenta receitas organizadas em um layout simples, utilizando estrutura HTML e estilização com CSS.

Este projeto faz parte das aulas que estou realizando para aprendizado em desenvolvimento **Front-End**.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

## 🎯 Objetivo

Praticar:

* Estruturação de páginas com HTML
* Estilização com CSS
* Organização de conteúdo
* Boas práticas iniciais em desenvolvimento web

## 📂 Como visualizar o projeto

1. Baixe os arquivos do repositório
2. Abra o arquivo `index.html` em seu navegador
